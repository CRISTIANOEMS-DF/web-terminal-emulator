## 🎯 How It Works

### Theme Selection
```
User: window.ConsoleTheme.set('matrix')
  ↓
ConsoleTheme updates internal state
  ↓
Emits 'theme:changed' event
  ↓
CSSLinkage listens and injects new variables
  ↓
All elements using var(--console-*) update instantly
  ↓
WebSocket broadcasts to other clients
```

### Real-time Synchronization
```
Client 1: Changes theme
  ├─ Updates local CSS
  ├─ Broadcasts via WebSocket
  │
Client 2: (Connected)
  ├─ Receives broadcast
  ├─ Updates their CSS
  │
Client 3: (Connected)
  ├─ Receives broadcast
  └─ Updates their CSS
```
